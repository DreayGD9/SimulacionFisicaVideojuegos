
# CONTROLES

J - Acelerar izquierda

L - Acelerar derecha

K - Ralentizar aceleración

WASD - Cámara

ESC - Salir del juego


El juego consiste en evadir proyectiles que te lanza un enemigo desde el fondo (empieza después de 5 segundos), intentando mantenerte sobre una cuerda floja.
EL juego termina cuando caes de la cuerda.
